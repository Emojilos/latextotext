# LaTeX to Text

## Что это
- Одностраничный фронт (`index.html`) с темной темой: ввод LaTeX, живой рендер (MathJax), поле вывода с сохранением переносов.
- Двухзонная верхняя панель: 75% под ввод, 25% под инструменты; нижняя карточка — рендер во всю ширину.
- Работает целиком в браузере, без бэкенда и шаблонов.

## Быстрый старт
```bash
# вариант 2: локальный статический сервер (чтобы обойти CORS на некоторых браузерах)
python3 -m http.server 8000
# затем открыть http://localhost:8000/index.html
```

## Структура
- `index.html` — UI, стили, логика рендера, загрузка/сохранение/сниппеты.

## Примечания
- MathJax грузится с CDN (интернет обязателен).
- Автосохранение включено по умолчанию (localStorage).
- Поддержка drag & drop `.tex` / `.txt` прямо на дропзону.
```
